This directory contains Crysterm tests, sorted by source or type.

Tests are less functional than examples, showing only individual
Crysterm widgets, features, and their interaction.

The widget/ and layout/ tests are automatically included in the
documentation at appropriate places. Other tests are comparisons/
equivalents of other toolkits' tests and/or other tests Crysterm
maintains.

For more functional code, see examples/.
For more low-level code, see spec/.
